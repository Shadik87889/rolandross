Organizing Your Services

Category Generation

Admin Term: Sector Taxonomy / Sub-Segments
Simple Meaning: Main Service Folders (e.g., Hair Care, Spa) and Sub-Folders (e.g., Hair Cut, Facial).

Before adding a service, you must create a Category to put it in. Think of these as the main chapters of a book.

Step-by-Step Instructions

1 Click the Formulate Sector button (has a '+' icon).
2 Paste an image link into the Direct Image URL box. The system will automatically show you a preview.
3 Type the Category Name & Subtitle in English (left side) and Bengali (right side).
4 To create sub-categories (like "Hair Color" inside "Hair Care"), click Add Segment and type the names.
5 Click Establish Sector to save it.

How to Change the Order

Don't like the order categories appear on your website? Click the Reorder Taxonomy button. You can simply drag and drop the items into the perfect order, then click Lock Sequence.

Adding Treatments

The Vault (Services)

Admin Term: Vault Signature / Inject New Service
Simple Meaning: Creating a new Treatment/Service to show on the website.

This is where you add the actual salon treatments. You must link every service to a Category you created in the previous step.

Step-by-Step Instructions

1 Click the Inject New Service button.
2 Paste your high-quality image link into the Image URL box.
3 Assign Sector: Click the dropdown and search for the Main Category this service belongs to.
4 Lock Segment: If you created sub-categories earlier, select the specific sub-category here.
5 Enter the Service Name and Description in both English and Bengali.
6 Click Commit to Vault.

* Note: You can use the search bar to quickly find existing services, and the pen icon to edit them anytime.

Your Portfolio

The Gallery

Admin Term: Visual Assets / Inject Rendering Layer
Simple Meaning: Uploading photos to your website's Gallery page.

Step-by-Step Instructions

1 Click the Inject New Image button.
2 Paste your image link.
3 Provide an Image Title and Subtitle in both languages.
4 System Filter Mapping: This is highly important! Choose whether this photo belongs to "Aesthetic Clinic", "Luxury Spa", or "Products". This controls which tab the user clicks on the website to see the image.
5 Click Commit Visual.

Advanced Features

Promotional Matrix (Offers)

Admin Term: Algorithmic Target Vector / Neural-Sync
Simple Meaning: Creating Discount Codes, Live Countdowns, and Custom Bundles (like a Bridal Package).

This is the most powerful tool in your entire command center. It generates live scrolling offer cards on your home page with working countdown timers!

How to Create a Promotion

1 Click Formulate Offer.
2 State & Validity: Set the status to "Active" and pick an Expiration Date. The website will automatically turn this into a live ticking clock!
3 Enter your Campaign Title (e.g., "EID Supreme Package") and an optional Promo Code.

Step 4: The Target (Crucial Step)

You must tell the system WHAT is on discount. Click the dropdown to choose:

  • Single Service / Category: Choose this to discount just one item.
    (Bonus: The system uses "Neural-Sync" to automatically pull the image for you! You don't even need to upload a photo.)
  • Custom Bundle (Multi-Select): Want to create a "Bridal Package"? Select this! A giant checklist will appear, allowing you to combine multiple services (like Haircut + Facial + Spa) into one mega-offer.
5 Financial Matrix: Choose either "Percentage %" or "Flat Value ৳" and enter the amount.
6 Click Establish Matrix to launch the campaign live.

System Protection

Security Protocol

To keep your digital empire safe from unauthorized access, you can update your password at any time.

1 Navigate to the Security Protocol tab in the left menu.
2 Enter your new password into the New Master Cipher box (it must be at least 6 characters long).
3 Click Execute Override. Your credentials will update instantly.

Never share your Access Cipher (Password) with anyone. If you lose access, contact your web engineering team immediately to force a database reset.
